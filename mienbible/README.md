# Mien Bible Website - Migrated for savemightlife.com

This directory contains the website files migrated to be hosted under the `savemightlife.com/mienbible/` path.

## Deployment Instructions

1.  **Upload Files**: Upload the entire contents of this directory (all files and folders) to the web server directory that is configured to serve the `/mienbible/` URL path on `savemightlife.com`.

2.  **Example**: If the document root for `savemightlife.com` is `/var/www/html`, you would place the contents of this folder into `/var/www/html/mienbible/`.

3.  **Access**: The website's main page will then be available at `http://savemightlife.com/mienbible/index.html`.
